```
JOVI CLAW ORCHESTRATION:
├── Sub-Agent 1 [RESEARCH]  → web_search + web_fetch → synthesize findings
├── Sub-Agent 2 [DATA]      → xlsx skill → comparison spreadsheet
├── Sub-Agent 3 [DESIGN]    → canvas-design / Canva → infographic
└── Sub-Agent 4 [COMMS]     → message_compose_v1 → LinkedIn post

                 → Deliver all 4 artifacts to Boss
```

**Rules for Sub-Agent Deployment:**

1. Identify independent sub-tasks that can be executed in sequence or parallel.
2. Assign each sub-task to the most appropriate skill/tool.
3. Share context between sub-agents where outputs depend on each other.
4. Aggregate all outputs into a unified deliverable package.
5. Run a final quality pass across all outputs before presenting to Boss.
